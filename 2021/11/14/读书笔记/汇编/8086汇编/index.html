第五章

一个二进制位为一个bit
字：记为”word”，16位二进制位为“字
字节：记为”byte”，8位二进制位为“字节””

idata表示常量——>mov ax,[1] <=> mov ax,[idata]
[bx]表示：

1.例：mov ax,[bx]
(ax)=((ds)*16+(bx))=(ax)

loop循环：[cx]操控着循环次数，先将[cx]减1，若[cx]不为零时，则执行标识位置代码；[cx]为零时，则向下执行。

   1.例：mov [cx],11
      s: add [ax],[ax]
       loop s
   2.即：mov [cx],循环次数
     s: 循环执行的程序段
      loop s
   注：汇编语言中，一个标号代表一个地址，程序中的s所标识的地址处执行

命令：

例: -g 0012，指命令debug执行至0012处
-p:指循环一次执行完，不再步步追踪循环内计算

masm编译器和debug的区别:

   1.对[1]:debug为((ds)*16+1)
           masm为1
   2.汇编asm源程序写法[1]--->ds:[1]，即应在偏移地址前，给出该内存的段地址

第九章 转移命令的原理

1.操作符 offset

   功能：获得标号的偏移地址
   例：mov ax,offset start        //指获得start的偏移地址

2.jmp指令
jmp需要给出两种信息：

   1)转移的目的地址
   2）转移的距离(段间转移、段内短转移、段内近转移)
   格式：a. jmp short 标号(转到表好处执行指令)
             注：i.这种格式实现的是段内短转移，对ip修改范围为-128~127
                ii.short指明此处为8位位移
          b.jmp near ptr 标号
               注：i.这种格式实现的是段内近转移，对ip修改范围为-32768~32767
                  ii.near指明此处为16位位移

由底层机器语言说明，CPU在执行jmp指令时不需要转移的目的地址，就可以实现对IP的修改

           原理：转移指令没有告诉CPU要转移的目的地址，但告诉CPU要转移的位移

1.ret和retf

2)retf

2.call指令

   操作：1)将当前的IP或CS和IP压入栈中

5.mul指令
mul是惩罚指令

       注：1)两个相乘的数，应都是8位或都是16位。若都是八位，一个默认放在al中，另一个放在>        位reg或内存字节单元中；如果是16位，一个默认在ax中，另一个放在16位reg或字节内
           存单元中。
           2)结果：若是8位乘法，默认放在ax中；如果是16位乘法，结果高位默认在dx中存放，低
           位在ax中存放

第十一章 标志寄存器

flag标志位：
有的指令对标志寄存器没有影响，如mov、push、pop
1.ZF标志位

       flag的第六位,零标志位，用来记录相关指令执行完后是否为零。
       **如果为零，zf=1;否则zf=0**

2.PF标志位

       flag的第二位是PF，奇偶标志位，**用来记录所有比特位中1的个数是否为偶数**
       **如果1的个数为偶数，pf=1;否则为0**

3.SF标志位

           flag的第七位是SF，符号标志位。**用来记录相关指令执行后，其结果是否为负。**
           **如果结果为负，sf=1;如果非负，sf=0**
                                                           
           ![在这里插入图片描述](D:\Typora\file picture\20210115145901251.png)

![image-20210605223738169](D:\Typora\file picture\image-20210605223738169.png)

4.CF标志

           flag的第零位是CF，进位标志位。一般情况下，在记录**无符号数**运算时，它记录了运算结果的
           最高有效位向更高位的进位值，或从更高位的借位值

5.OF标志

               flag的第十一位是OF，溢出标志位。一般情况下，OF记录了**有符号数**运算的结果是否发生了
               溢出。
               **如果发生溢出，OF=1；如果没有，OF=0**

6.补码

           1）补码取反加1得原数得绝对值
           2）原数取反加一得补码，
           3）标识数正负看补码
           5）正数补码为原码本身，负数补码为原码取反加一
           例：F0H为-16的补码

7.adc指令

                           1）是带进位加法指令，利用到了CF位上记录的进位值

8.sbb指令

                           1）带借位减法指令，利用了CF位上记录的借位值

9.cmp指令

           1）cmp是比较指令，cmp的功能相当于减法指令，只是不保存结果

10.检测比较结果的条件转移指令
| 命令 | 含义 | 检测的相关标志位 |
| ——– | —–: | :—-: |
| je | 或等于则转移 | zf=1 |
| jne | 不等于则转移 | zf=0 |
| jb | 低于则转移 | cf=1 |
| jnb | 不低于则转移 | cf=0 |
| ja | 高于则转移 | cf=0且zf=0 |
| jna | 不高于则转移 | cf=1或 zf=1 |

11.DF标志和传传送指令
df=0，每次操作后si和di递增
df=1，每次操作后si和di递减
cld指令：将标志寄存器的df位置0
std指令：将标志寄存器的df位置1

传传送指令
movsb指令：指将ds:si指向的内存单元中的字节送入es:di中，然后根据标志寄存器df位的
值，将si和di递减或递增
movsw指令：指将ds:si指向的内存单元中的字送入es:di中，然后根据标志寄存器df

    设置IF指令：
    sti，设置IF=1
    cli，设置IF=0
